# Full-Stack Task & Directory Management App 

A professional web application for managing tasks and directories, built with a focus on modern state management and responsive design.

##  Tech Stack
- **Frontend:** React.js
- **State Management:** Redux Toolkit 
- **Backend:** Node.js & Express
- **Database:** MongoDB (Mongoose)
- **Design:** Fully Responsive UI

##  Key Features

- **Advanced Task Management:** Full CRUD operations (Create, Read, Update, Delete) for tasks.
- **Dynamic Directory System:** Complete CRUD functionality for directories, allowing users to create, rename, and manage custom categories for tasks.
- **Advanced State:** Centralized data handling using Redux Toolkit for seamless navigation and instant UI updates.
- **Cross-Device:** Fully optimized and Responsive UI for mobile, tablet, and desktop screens.  


##  How to Run

1. **Server:**
   - `cd server`
   - `npm install`
   - `npm run dev`

2. **Client:**
   - `cd client`
   - `npm install`
   - `npm run dev`
